## macbook 闲置发烫解决

### 发热症状

每次第二天上午打开电脑时，电脑很烫，似乎运行了很久的高cpu程序。但明明闲置了一晚上，啥都没有干，连续几天都一直发烫无法解决。设置了好几次终于解决！

我的电脑型号：`MacBook air M2 24+512G` `Sequoia 15.5 `

### 排查步骤

- 打开活动监视器程序，点击能耗一栏，找到占用能耗最高程序

- 如有AIDent软件设置电量，可以查看24小时以内的电脑温度的变化

- 打开终端，输入如下命令：查看是哪些进程频繁唤醒电脑，将输出的所有内容粘贴给GPT，它会给你分析

  ```shell
  pmset -g log | grep -e "Sleep.*due to" -e "Wake.*due to"
  ```

**我的电脑问题：**

- 聚焦占用最多的能耗
- 电脑温度频繁上升多次
- 蓝牙wifi频繁唤醒、media-analyised程序频繁唤醒吃cpu

### 问题分析

综上,我的电脑是在插电盒盖状态下，闲置一直库库发烫。

macOS 的逻辑是：

> 插电 = 不心疼电
>  合盖 = 不打扰你
>  那我就疯狂干活 

问题：

1.我之前打开了共享和蓝牙以及隔空传送与接力，所以一直试图与我的平板建立连接

2.聚焦一直在高cpu运行，建立文件索引spotlight，并且media-analyised一直在用神经网络分析照片文件

3.icoud等其他程序、查找我的mac等程序时不时唤醒

### 解决步骤

#### 1.关掉接力和共享

打开设置：通用>隔空传送与接力，通用>共享，不需要的都直接关了，或者用的时候再打开

#### 2.关掉聚焦分析

打开设置，找到聚焦，滑到下面找到“搜索隐私”

如果像我一样聚焦轻度使用，只用于打开程序app，直接将/Users/你的用户名拉进，防止聚焦搜索。

或者平时高强度使用聚焦，那就将那些复杂文件夹，比如代码仓库、成千上万的照片文件夹直接拉入。

#### 3.终端设置

打开终端分别输入两次如下命令,如果需要输入密码就输入你的开机密码，然后回车

```shell
sudo pmset -a powernap 0 # 用于设置电脑小憩否
```

再输入命令回车：

```shell
sudo pmset -a networkoversleep 0 # 熄屏了就不使用网络
```
